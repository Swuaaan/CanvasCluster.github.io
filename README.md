# CanvasCluster

This project is a simple HTML and JavaScript application that allows you to draw an array of polygons or symbols on an HTML canvas. It utilizes modular JavaScript components for better code organization.

This project was inspired by [Sebastian Hahner](https://github.com/sebinside/).
